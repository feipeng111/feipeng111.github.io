关机、重启、退出：
reboot :重启
init 6 :重启
shutdown -r :重启
init 0 :关机
shutdown -h now :关机
halt :关机
poweroff :关机
logout :退出
exit :退出

几种系统模式 init :
level 0 :关机
level 3 :命令行模式
level 6 :重启

./或/ 代表根目录

sudo : 以管理员权限执行
sudo passwd root :改密码
su :切换账户
clear : 清屏 或Ctrl+L
ifconfig : 可设置网络设备的状态，或是显示目前的设置
init 3 :切换到命令行模式
startx ：启用桌面

查看文件：
cat

ls :列出目录
*
-a :全部的文件，连同隐藏档( 开头为 . 的文件) 一起列出来(常用)
*
-d :仅列出目录本身，而不是列出目录内的文件数据(常用)
*
-l :长数据串列出，包含文件的属性与权限等等数据；(常用)

pwd :显示目前所在目录
*
-P :显示出确实的路径，而非使用连结 (link) 路径。

more :让画面在显示满一页时暂停，此时可按空格健继续显示下一个画面，或按 q 键停止显示。
less :less 命令的用法与 more 命令类似，也可以用来浏览超过一页的文件。所不同的是 less 命令除了可以按空格键向下显示文件外，还可以利用上下键来卷动文件。当要结束浏览时，只要在 less 命令的提示符“：”下按 q 键即可。
tail :显示后几行
head : 显示前几行

文件、文件夹：

mkdir [-mp] : 新建文件夹
*
-m :配置文件的权限 mkdir -m 755 test
*
-p :帮助你直接将所需要的目录(包含上一级目录)递归创建起来 mkdir test1/test2…

rm [-fir]:删除文件
*
-f :就是 force 的意思，忽略不存在的文件，不会出现警告信息
*
-r :递归删除！最常用在目录的删除！这是非常危险的选项
*
-i :互动模式，在删除前会询问使用者是否动作

cp :复制文件或文件夹到指定位置
*
cp -r ../Documents . : -r指复制目录，. 指当前目录， 可以使用相对路径或绝对路径来复制

mv :移动文件或文件夹到指定位置
touch :新建文件

rmdir :删除文件夹
*
-p :连同下一级『空的』目录也一起删除 如：rmdir -p w s ss (w、s、ss为空目录)

cp :复制文件或文件夹到指定位置
*
cp -r ../Documents . : -r指复制目录，. 指当前目录， 可以使用相对路径或绝对路径来复制

chomd : 更改文件权限

查找文件:
find :全盘查找文件
grep :全盘所有文件中查找某些字符，或者在某个文件中查找，通过ls 目录 |grep ‘查找内容’ 也能找到包含查找内容的文件名
*
find和grep处理对象不同，find是在文件夹中找文件用的，可以针对文件属性（大小、日期等）进行约束；
*
grep本质是查找文件中的字符串的，可以用正则表达式，功能强大。

whereis :查找某个命令的所在位置
which :查找当前用户的命令
*
cd 这个常用的命令竟然找不到啊！为什么呢？这是因为 cd 是bash 内建的命令！ 但是 which 默认是找 PATH 内所规范的目录，所以当然一定找不到的！

进程:
jobs :显示当前暂停的进程（与ctrl+z(暂停)配合）
bg %N :使第N个任务在后台运行
fg %N :使第N个任务在前台运行
默认bg，fg不带%N时表示对最后一个进程操作！
*
jobs命令只看当前终端生效的，关闭终端后，在另一个终端jobs已经无法看到后台跑得程序了，此时利用ps（进程查看命令）

ps -aux :查看进程
*
-a :显示所有程序
*
-u :以用户为主的的格式来显示
*
-x :显示所有程序，不以终端机来区分

kill 进程号 ：关闭进程
& :在命令最后加上&符号，表示让这个进程到后台去执行，这样立刻返回到提示符状态，我们可以接着做下面的事。如：command &
*
& 表示让这个进程到后台去执行,如果关了shell窗口甚至退出ssh登录或vnc登录，那么进程自动就结束了

nohup :退出窗口乃至退出登录仍然保持程序运行
*
例nohup command &

压缩文件:
压缩包格式有：.tar .gz .tar.gz .tgz .bz2 .tar.bz2 .bz .tar.bz .Z .tar.Z .xz .zip .rar .7z
常用格式 .tar.gz 和.zip他们的解压和压缩命令如下：
.tar :
压缩 : tar cvf xxx.tar.gz wsxxx
解包 : tar xvf xxx.tar.gz 解包后原始文件依然存在

.zip :
压缩 : zip hhh.zip xxx(要压缩的文件名)
解包 :unzip xxx.zip
列出内容 : unzip -l xxx.zip

vi/vim工作模式：
切换到插入模式：
*
i，I ：i为[从目前光标所在处插入]，I为[在目前所在行的第一个非空格符处开始插入]
*
a，A : a为[从目前光标所在的下一个字符处开始插入]，A为[从光标所在行的最后一个字符处开始插入]
*
o，O : 英文o为[在目前光标所在得下一行处插入新的一行]，O为[在目前光标所在处的上一行插入新的一行]

末行模式：
set nu : 设置行号
w ：保存内容
wq ：保存内容并且退出编辑器
q! ：不保存内容直接退出编辑器

命令行模式-搜索：
/word : 向光标之下寻找一个名称为word的字符串
n :代表重复前一个搜寻的动作
:%s/word1/word2 : 寻找word1这个字符串，并将该字符串取代为word2

命令行模式-删除，复制粘贴：
x，X ：在一行字当中，x为向后删除一个字符，X为向前删除一个字符
dd ：删除光标所在的那一行
ndd ：n为数字，删除光标所在的向下n行
yy ：复制光标所在的一行
p ：将复制的一行粘贴在光标的下一行
P ：将复制的一行粘贴在光标的上一行
u ：复原前一个动作
